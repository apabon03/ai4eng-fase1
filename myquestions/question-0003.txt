Clasificación con Stacking (Sklearn)
El Problema:
A veces un solo modelo no es suficiente. El "Stacking" permite usar las predicciones de varios modelos como entrada para un modelo final (meta-modelo) que toma la decisión definitiva.

Tu Misión:
Escribe una función llamada treinar_modelo_stacking(X, y) que:

Configure un StackingClassifier que use dos estimadores base: un RandomForestClassifier y un KNeighborsClassifier.

Use una LogisticRegression como estimador final (final_estimator).

Entrene el ensamble con los datos X e y y retorne el modelo entrenado.

Entrada: X (numpy array o DataFrame), y (numpy array o Series).
Salida: Objeto StackingClassifier entrenado.
